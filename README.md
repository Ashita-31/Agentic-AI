## 🌟 Overview

**Agentic AI** represents the next evolution of artificial intelligence — systems that can **think, plan, and act autonomously**.

This project demonstrates how to build intelligent agents that:
- 🧠 Reason through problems  
- 📋 Plan multi-step actions  
- ⚡ Execute tasks using tools  
- 🔁 Learn from interactions  

---

## 🧠 What is Agentic AI?

Agentic AI refers to AI systems that behave like **autonomous agents**, capable of:

✔ Goal-oriented decision making  
✔ Multi-step reasoning  
✔ Tool usage (APIs, search, etc.)  
✔ Memory retention  

---

## ⚙️ Architecture

```bash
User Input → LLM (Reasoning) → Agent Planner → Tool Execution → Memory → Output# Agentic-AI
```

## 🧪 Example Usage
``` PYTHON
from langchain_groq import ChatGroq

agent = ChatGroq(model="llama-3.3-70b-versatile")

response = agent.invoke("Plan a trip to Goa with budget constraints")

print(response.content)
```
